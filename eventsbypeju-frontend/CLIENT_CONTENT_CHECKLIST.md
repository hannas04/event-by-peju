## 📸 FILE DELIVERY SPECIFICATIONS

**Image Quality Standards:**
- Resolution: Minimum 1200x1200px (preferably 2000x2000px or higher)
- Format: JPG (compressed but high quality) or PNG
- File Size: Keep under 5MB per image (for web optimization)
- Color Profile: RGB (not CMYK)

**File Naming Convention:**

- `gallery_02_corporate_gala.jpg`
- `hero_portrait_peju.jpg`
- etc.
